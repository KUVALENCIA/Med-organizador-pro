# 👤 Organizador Médico Pro - Cuidadores

## 🏥 Propósito y Funcionalidad
Esta aplicación web está diseñada específicamente para cuidadores de pacientes médicos o enfermeras que necesitan gestionar múltiples tratamientos de forma ordenada. Permite el registro flexible de medicación para **varios pacientes**, la toma directa de fotos de los fármacos con la cámara del celular, y genera reportes profesionales en formato **PDF**, separando a cada paciente en una hoja nueva para su fácil distribución o envío por WhatsApp/email.

### Opción 2: "Instalar" en tu Pantalla de Inicio (Android/iOS)
Para usarla como una aplicación nativa con su propio ícono:
1. Abre el enlace web en tu celular (Chrome para Android o Safari para iOS).
2. Toca el menú de opciones (Chrome: tres puntos arriba / Safari: icono de compartir abajo).
3. Selecciona **"Añadir a pantalla de inicio"** (o similar).
4. Verás el ícono profesional en tu menú y la aplicación se abrirá sin barras de navegación para una experiencia completa.

---

## 📝 Características Técnicas
* **Agrupación Automática:** Organiza medicamentos por nombre de paciente.
* **Toma Directa:** Botón especializado para abrir la cámara del celular.
* **Flexibilidad de Dosis:** Permite ingresar fracciones (1/2, un cuarto) y períodos de tiempo (semanas, meses).
* **Seguridad:** Botones de confirmación para eliminar datos.
